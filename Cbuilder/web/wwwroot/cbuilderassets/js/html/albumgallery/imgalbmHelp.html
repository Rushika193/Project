• 1. Drop the component.
  • 2. Choose desired Album Layout or close the Layout popup that result in default layout.
  • 3. Manage Album and Images from the popup or from the Manage ImageAlbum option.
  • 4. Double click Album in Album List Popup to manage images of that Album.
  • 5. Landing page can be selected from the setting. Current page is the default landing page.
  • 6. Clicking any Album will redirect to the Image Gallery page of that Album.
  • 7. If landing page other than default is used, after redirecting to that page, the component should be dropped again.
  • 8. Images can be managed from Image Gallery page, which will update the thumbnails in Album Layout.
  • 9. Image Layout can be changed from the drop down of basic setting tab.
  • 10. Save Image Gallery after designing the layout.

  • NOTE:Make sure to save the page after applying settings to image gallery.
