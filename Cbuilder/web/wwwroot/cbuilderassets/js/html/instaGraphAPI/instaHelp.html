1.Instagram business account is required to use FB Graph API for instagram So, connect a Facebook page with your instagram account first.
    2. Go to Facebook App and create a Facebook app and setup Instagram.
    3. After you have successfully created Facebook app setting up Instagram, Go to Graph Explorer.
    4. At the right of Access Token field, there is a drop down. Select get user access token.
    5. This Access token is short lived. For a long lived access token you can follow up on Facebook documentaion.
    6. You will require this access token and the url to your Facebook page(i.e linked to your Instagram) to get your instagram picures.
    7. In the basic tab of setting, submit with access token, facebook page url and number of pictures to fetch(max 20) per request. Hit the submit button.
    8. Upon successful API connection, your instagram photos will be displayed in the view in Grid layout.
    9. Then you can tweak your photos to your liking.
