To Generate google API key for Youtube Data API v3.
1. Go to Get API Key.
2. Create a project for requesting an API key
3. Under Dasboard Tab Click Enable API Services.(Make sure that API key for Youtube Data API v3 is enabled.)
4. Search for Youtube Data API v3 and click on ENABLE button.
5. On question"Which API are you using?" Choose Youtube Data API v3(which will be selected by default).
6. On question"Where will you be calling the API from?" Choose Web browser(Javascript).
7. On question"What data will you be accessing?" Choose Public Data.
8. Then Click on "What credentials do I need?" button.
9. You will be provided with the API key. Save it for later use in the component.
10. You can put restrictions on your API key from the credentials tab.

Steps to use youtube playlist component.
1. Go to Youtube.
2. Search for a youtube channel and copy its url by right clicking on the channel name.
3. Now paste the API, the channel URL and the request limit (which is 50 max) in the respective text box.
4. Click on Submit button.
